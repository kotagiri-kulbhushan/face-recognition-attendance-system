# Face Recognition Attendance System

A real-time face recognition based attendance system built using **Flask** and **OpenCV**.  
The system supports employee registration, face-based check-in and check-out, and automatic attendance logging.

## Features
- Register new employees using face capture
- Automatic model training
- Face-based check-in and check-out
- Unknown face detection and blocking
- Real-time camera feed in browser
- Attendance stored in CSV format

## Tech Stack
- Python
- Flask
- OpenCV (LBPH Face Recognizer)
- HTML & CSS
